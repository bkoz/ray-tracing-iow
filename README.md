# ray-tracing-iow
Code exercises for Pete Shirley’s “Ray Tracing in One Weekend”

### Build
```
mkdir build
cd build
cmake ..
make
```

### Run

The image `convert` program is provided by **ImageMagick**.
```
./ray-tracer-main > image.ppm
convert image.ppm image.png
```
